# Currículo Profissional (Web Version)

Este projeto consiste em uma página web desenvolvida para servir como uma versão online do meu currículo. O foco principal foi a aplicação de conceitos básicos de HTML, além da utilização do **GitHub Pages** para hospedagem.

## Funcionalidades
* **Apresentação Profissional:** Resumo de qualificações, formação acadêmica e projetos.
* **Acesso Rápido:** Disponível 24/7 para recrutadores através de um link direto.
* **Design Responsivo:** O layout foi estruturado para ser lido de forma clara em navegadores desktop.

## Link de Acesso
O site está publicado e pode ser acessado em:
[**Clique aqui para abrir o Currículo Online**](https://claytonluisrama.github.io/meu-portfolio/curriculo-web/)

## Tecnologias Utilizadas
* **HTML:** Estruturação do conteúdo.
* **Git/GitHub:** Controle de versionamento.
* **GitHub Pages:** Serviço de deploy e hospedagem.
